Security & Compliance

GMP-ready. Audit-first. No shortcuts.

Nexus was designed with pharmaceutical compliance in mind — every action is traceable, every record is immutable.

SHA-256 Audit Hash

Every activity log entry is cryptographically hashed. Any tampering is mathematically detectable — immutable audit trail by design.

Row-Level Security

Every database table enforces Supabase RLS policies. No user can read or write data outside their organization — enforced at the database level.

Role-Based Access

7 distinct roles with scoped permissions: BD, PM, Dept Member, Dept Head, Upper Management, PM Head, and Super Admin.

Complete Org Isolation

Your organization's data is completely isolated from every other organization — at API, database, and storage layers.

Document Malware Scanning

All BD uploaded documents are scanned for malware, macro-embedded executables, and magic byte mismatches before storage.

Status Machine Enforcement

Proposal status can only advance through defined transitions — no user can skip steps or bypass approval gates, even via direct API calls.

Complete Activity Log

Every action — submissions, approvals, edits, logins, rejections — logged with actor, timestamp, department, and payload.

Dept Cross-Write Prevention

A Quality user cannot modify Manufacturing data, even if they know the endpoint. All writes enforce department scoping server-side.

Zero Client-Side Secrets

No service keys, admin credentials, or sensitive data ever reaches the browser. All privileged operations run exclusively on the server.
